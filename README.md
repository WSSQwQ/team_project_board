# Team Project Board

班级项目协作看板 - PySide6 GUI 项目

## 运行方式

```bash
pip install -r requirements.txt
python main.py
```

## 项目结构

- `main.py` - 程序入口
- `ui/main_window.py` - 主窗口UI
- `data/project_info.py` - 项目信息（组员A修改）
- `data/members.py` - 成员列表（组员B修改）
- `data/features.py` - 功能清单（组员C修改）
- `data/progress.py` - 进度追踪（组员D修改）
- `data/changelog.py` - 版本日志（组员D修改）
